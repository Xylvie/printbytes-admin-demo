# PrintBytes Admin Dashboard

An admin dashboard for a local printing shop to manage sales, expenses, and net profit. Visualize daily and monthly metrics with interactive charts and track business performance in real time.

## Features
- Add daily sales, expenses, and partitions
- View total overall sales, monthly sales, expenses, and net profit
- Compare current vs previous month with Chart.js charts
- Responsive interface built with TailwindCSS

## Tech Stack
- Laravel
- TailwindCSS
- Chart.js
- JavaScript

## Usage
Log in to the dashboard to manage sales and expenses

View charts to track business performance
